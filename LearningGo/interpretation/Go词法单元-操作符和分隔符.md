#### 操作符和分隔符

---

- 算数运算符

```
+ - * / %
```

- 位运算符

```
& | ^ &^ >> <<
```

- 赋值和赋值复核运算符

```
:= = += -= *= /= %= &= |= ^= &^= >>= <<=
```

- 比较运算符

```
> >= < <= == !=
```

- 括号

```
() {} []
```

- 逻辑运算符

```
&& || !
```

- 自增自减

```
++ --
```

- 其他运算符

```
: , ; . ... <-
```
